# Frontend - Real Estate Platform

React + TypeScript frontend for the Real Estate Platform.

## Installation

```bash
npm install
```

## Running the app

```bash
# development
npm run dev

# build for production
npm run build

# preview production build
npm run preview
```

## Environment Variables

See `.env.example` for required environment variables.

## Features

- Browse properties, lands, and projects
- View detailed information for each listing
- Responsive design
- TypeScript for type safety
- Vite for fast development

## Tech Stack

- React 18
- TypeScript
- Vite
- React Router
- Axios
