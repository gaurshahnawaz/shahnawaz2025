# Sample Data - Images

This directory contains placeholder images for the Real Estate Platform.

## Directory Structure

```
images/
├── properties/     # Property images
│   ├── property1.jpg
│   └── property2.jpg
├── lands/          # Land images
│   ├── land1.jpg
│   └── land2.jpg
└── projects/       # Project images
    ├── project1.jpg
    └── project2.jpg
```

## Note

In a production environment, these would be actual JPEG image files. For the assessment, these are placeholder text files. Replace them with actual images as needed.

## Image Guidelines

- **Properties**: Photos of villas, apartments, and townhouses
- **Lands**: Aerial or ground-level photos of land parcels
- **Projects**: Renderings or photos of development projects

Recommended image dimensions:
- Listing cards: 800x600px
- Detail pages: 1200x800px
- Format: JPEG or PNG
